A little bit about myself

Since I was very young, I've always been passionate about technology and science. From the very beggining, I've been playing around with computers running on systems such as Windows 98 to Windows 11, a few different distributions of GNU/Linux since 2007, and macOS since 2009.

In 2020 I started my carrer path to become a professional software developer, being my first programming language Swift which I learned by myself following Apple's official documentation and using some resources such as Playgrounds and some ebooks. Then I've learned some of OOP with C++ using Visual Studio as the main IDE.

In 2021, while I was working in the Construction Industry, I decided to start new official studies of Multiplatform Applications Development, at EFA El Campico in Jacarilla (Alicante). This last year I learned about MySQL, HTML, CSS, JavaScript, Java, git, docker, Apache Servers, IntellijIDEA, etc.

While there's plenty of new stuff I'll come around learning in the future, I know that I have the knowledge, energy and enthusiasm needed to be part of amazing projects and to create great applications for people.
